# grito-web

Marketing website for Grito — AI sports coaching apps under the JDM Studios LLC brand.

Production: https://gritoai.com

## Stack

- Static HTML (self-contained, all assets embedded)
- Served via `serve` on Railway
- Custom domain managed in Railway → DNS at Squarespace

## Deploy

Auto-deploys from `main` branch via Railway.

## Local preview

```bash
npx serve .
```

Then open http://localhost:3000
